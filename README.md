### Key Technical Projects

#### [hive-transcendence](https://github.com/erkkaervice/hive-transcendence)
*Full-Stack Architecture & Real-Time Systems*
- **The Challenge:** Architecting a real-time, multi-user web ecosystem with centralized authentication and a containerized backend.
- **The Solution:** Developed a Single Page Application (SPA) utilizing **Next.js** and **MariaDB**, orchestrated via **Docker**.
- **Competency:** Full-stack integration, real-time data synchronization, and containerized deployment.

#### [hive-webserv](https://github.com/erkkaervice/hive-webserv)
*Network Engineering & Protocol Implementation*
- **The Challenge:** Managing concurrent client connections and high-traffic throughput without high-level frameworks.
- **The Solution:** Engineered a functional HTTP/1.1 server focusing on non-blocking I/O and I/O multiplexing (**select/poll**).
- **Competency:** Strict adherence to RFC standards, socket programming, and low-level network performance optimization.

#### [.dotfiles](https://github.com/erkkaervice/.dotfiles)
*Infrastructure as Code & Environment Hardening*
- **The Challenge:** Ensuring environment parity and security across disparate Linux distributions (Arch, NixOS, Debian).
- **The Solution:** Automated configuration management and **GPG/SSH hardening** via custom shell scripts and declarative configs.
- **Competency:** Security-first configuration management and automated environment provisioning.

#### [hive-inception](https://github.com/erkkaervice/hive-inception)
*Container Orchestration & System Administration*
- **The Challenge:** Orchestrating a secure, multi-tier web stack with persistent data and network isolation.
- **The Solution:** Engineered a multi-container Docker infrastructure including **NGINX**, **WordPress**, and **MariaDB** on isolated Alpine/Debian nodes.
- **Competency:** Network isolation, volume persistence, and secure service orchestration.
